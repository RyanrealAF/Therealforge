First Tenet: Purposeful Minimalism

Every element must earn its place. Clutter is avoided to maximize learning clarity and focus. This design philosophy translates into high-contrast text, deliberate spacing, and the avoidance of extraneous visual noise.

Second Tenet: Contrast & Control

Utilize dark backgrounds (charcoal, matte black) with controlled warm highlights (bronze, beige) to reduce eye strain and establish a focused, elegant atmosphere.

Toolstack (Resource Guide Compliance)

The entire workflow uses only free, open-source, or permissively licensed tools to maintain cost-free development and privacy.

  1. **VS Code:** Default editor for all code creation and management.
  2. **GitHub:** Primary resource for version control and free static hosting (GitHub Pages).
  3. **Chrome DevTools:** Essential for debugging, performance testing, and ensuring mobile-first responsiveness.

Document Creation Pipeline (PDF Assets)

Creating and managing official documentation assets is integrated into the free tool workflow, ensuring consistency and adherence to the \textbf{Dark Elegance} typography.

Recommended Free Toolchain:

**Typing/Writing**
Use **LaTeX** (via VS Code extensions or an online editor like Overleaf) or **LibreOffice Writer** (Free/Open Source) to draft the document content. The **LaTeX** environment is recommended for creating highly structured, formal documents like your strategic doctrine.
**Styling/Export**
When using LaTeX, the preamble ensures professional formatting and clean typography. For other tools, export directly to a high-quality PDF format, minimizing file size to maintain website performance.
**Deployment**
All final $\texttt{.pdf}$ files must be saved in the dedicated $\texttt{documents/}$ folder within the project root. This ensures a reliable relative link on GitHub Pages.
